## サンプル

https://ishi720.github.io/pake-style/

## インストール

- npm

```bash
npm i pake-style 
```

- yarn

```bash
yarn add pake-style 
```

## 読み込み

HTMLで以下のようにCSSファイルを読み込んで下さい

```html
<link href="pake-style/css/pake-style.min.css" rel="stylesheet" type="text/css">
```

## 使い方

- アニメーションぷるん(pake-anime-prun1s ~ pake-anime-prun5s)

指定した順に「ぷるん」の動作をします

